Pizzaria Rossi - Site de Delivery

Este é um site focado em delivery e venda rápida para pizzarias. O fluxo prioriza a rapidez: o cliente vê o total do pedido antes de finalizar e usa o WhatsApp do restaurante para combinar o meio de pagamento e os detalhes finais da entrega.

Principais características
- Menu enxuto para seleção rápida de produtos.
- Cálculo automático (ou a combinar) do valor do frete.
- Botão de finalização via WhatsApp para fechamento rápido do pedido.

Deploy
- Projeto pronto para publicar via GitHub Pages a partir da branch `main`.

Observações
- O WhatsApp é usado apenas para combinar pagamento/entrega; o valor a ser pago é exibido ao cliente antes de abrir o chat.
